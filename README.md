# Warm-Starting APSP with Predictions

This repository contains code accompanying paper titled *Warm-Starting All-Pairs Shortest Paths with Predictions* by Adam Polak and Jonas Schmidt (ESA 2026). It runs a simulation estimating the empirical prediction error on a real-world dataset.

## How to run it

```
g++ main.cpp -o main -O3 -std=c++17
./main
```
